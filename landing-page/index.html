Native Git Client

Full GUI for staging, committing, and branch management. Review diffs and discard changes without leaving the app.

Integrated Terminal

WebGL-powered xterm.js terminal. Split panes, persistent sessions, and full PTY support for running dev servers.

Local & Fast

Built with Rust and Tauri. Panes is a local-first application that feels native, snappy, and respects your data privacy.

Engine Agnostic

Bring your own intelligence. Connect to OpenAI, Anthropic, or local LLMs. Switch models per-turn.

Multi-Repo Workspaces

Manage complex monorepos or multi-service architectures. Context-aware chat that understands your file structure.

Approval Workflows

You are in control. Define trust levels for repositories and approve tool executions before they happen.
